Project : Pig Game

Goal: This game is for 2 players. The game starts with player 1.Player 1 roll a dice. Whatever the number on dice will get addd to the Current socre(current score for the current round). The player keep rolling the dice and the number gets added to current score. The player can Hold the score, on holding the score current score will get added to total score. The current score will become 0 now and we will switch the game to Player 2.

Player 2 can roll the dice. The number will gets added to current score first. If the player holds that score, it will be added to total score reseting the current score to 0.

Whenever a player roll the dice and gets 1, the player will loose all the current score and it's then the next player's turn. That's why sometimes it is necessory to hold our scores.

The first player to reach 100 point score will win the game. The game can be restarted by clicking on New Game button, which will resets the whole game.
