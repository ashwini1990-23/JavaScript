Project : Bankist(minimalist banking) App

Goal: Login with user: js and pin:1111
Online bank interface with all the movements(transactions), overall balance, account summary and 3 options to perform different actions like Transfer money, Request loan and Close account.

Request Loan: when user requests money, after some time it gets approved and it appears in movements as Deposit

Transfer Money: whenever user Transfers money, transfered money will get deducted from balance and it will be added to movements as withdrawl.

Account summary: we can sort movements, see account statistics

Close account: when user wants to close account, user needs to give his username and pin.
